# homebrew-cookunity

Homebrew tap for [cookunity-cli](https://github.com/sergiosegrera/cookunity-cli) — an unofficial CLI for CookUnity.

## Install

```bash
brew tap sergiosegrera/cookunity
brew install cookunity
```

## Usage

```bash
export COOKUNITY_EMAIL=your@email.com
export COOKUNITY_PASSWORD=yourpassword
cookunity user
cookunity menu
cookunity deliveries
```
